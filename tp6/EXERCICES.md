# Exercices pour la formation Python : TP6

## Création d'un package

A partir d'un des projets que nous avons mis en place lors de la formation : 
- Installer les dépendances nécessaires au build
- Créer la wheel de votre package
- Créer un nouveau venv, dans lequel vous installer votre wheel précédement crée
- Importer votre module pour l'utilser dans ce nouveau programme
